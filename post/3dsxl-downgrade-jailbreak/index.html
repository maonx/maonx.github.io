前些日子听我弟说他买了New 3DSLL，准备玩正版，虽然我自己也是玩正版，不过更喜欢破解后的汉化，这样可以玩中文游戏。因一直工作比较忙，都没有玩这个，也没看相关消息，听我弟这么一说，我就起性的搜索了一下，竟然发现有帖子说老版3DSXL可以无卡破解，就来了兴趣仔细研究了一下，现在手机的机子已经破解成功了，现在有空了记录一下破解日志。

无卡降级

什么叫无卡降级及破解

起初查帖子的时候我也一直有疑惑，到底这种方法需不需要GW、SKY之类的烧录卡来降级和破解了，等看了多篇帖子以后了解到这种方法完全不需要这些烧录卡了，直接用网上的攻略就可以降级及破解。

准备工具

  • 一台老版3DSLL（我的是美版），系统版本为9.2及以下；
  • 一张SD卡（最好大于4G，方便安装cia游戏，游戏体积都比较大，用默认SD卡的话注意备份）；
  • 一张R4卡（下面破解我用到的是R4卡，比较方便，而且本身我玩正版的时候也刚好有R4卡）；
  • 以下操作步骤请先保证有足够的电量，或直接插电源，至少不能操作当中断电。
  • 下载工具包 链接: http://pan.baidu.com/s/1dD35Sml 密码: mq3v

降级9.2系统到4.X

  1. 先将工具包中GW_Ultra_3.2.2(Public Beta)文件夹中的launcher.dat放到SD卡根目录。

  2. 3DS打开无线，用浏览器登录http://go.gateway-3ds.com 进入GATEWAY菜单（帖子说好多说是用扫码进入，不过我这一直都是失败，直接提示用浏览器打开此网址，我直接确定进入了）。

  3. 使用十字键的左右键选择BACKUP SAYSTEM NAND 进行本机NAND备份。备份完成之后会退回gateway菜单，此时直接按电源键关机，取出SD卡把你的NAND备份保存。（此NAND用于以后不小心把系统升到9.2以上版本救急用的） {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-8.jpg 400 %}

  4. 下载降级用固件，登陆http://www.gateway-3ds.com/downloads/ 在如图所示的位置下载主机区域所对应的固件CHN代表神游，JAP代表日版,USA代表美版,EUR代表欧版，TWN是港台版，KOR是韩版。下载以dg为后缀名的文件。将下载好的文件放在SD卡根目录里面。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-10.png 400 %}

  5. 将SD卡插加3DS，重新打开浏览器进入在线GATEWAY菜单，选择DOWNGRADE CONSOLE。会自动进行降级。降级后重启，我的美版3DS已经降到了4.2系统。 DOWNGRADE CONSOLE {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-15.jpg 400 %} 成功降至4.2 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-9.jpg 400 %}

制作EMUNAND（虚拟系统）

  • 首先将降级的4.2主机恢复出厂设置。
  • 将工具包中GW_INSTALLER.nds软件复制到R4内存卡。
  • 运行 GW_INSTALLER.nds及安装GATEWAY菜单 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-17.jpg 400 %} {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-1.jpg 400 %} {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-18.jpg 400 %}
  • 开机进入系统设置，利用传统的4-1-4方法，进入Gateway菜单。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-4.png 400 %}
  • 进入Gateway菜单，选择FORMAT EMUNAND 开始安装虚拟系统， 等运行完毕提示完成，长按电源键关闭3DS。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-16.jpg 400 %}

安装rxTools

  • 取出SD卡，检查你的SD卡中Nintendo 3DS\xxxxxxxxxxxxxx\xxxxxxxxxxxxxx\dbs （xxxxx为随机字母列）路径下是否有import.db和title.db文件，如果没有，自行创建相应的空文件。这一步很重要一定要检查清楚，不存在的话会导致无法安装rxTools，然后将SD卡插回3DS，启动进入设置图标->查看数据管理->3DS软件数据管理，3DS主机会提示文件损坏，并确认修复，自行修复生成正确的db文件。
  • 打开工具包中的rxTools v2.6,复制rxtools.dat、firmware.bin和slot0x25KeyX.bin文件到SD卡根目录（firmware.bin可由rxTools v2.6/tools/cdn_firm.py生成最新版本）；将rxinstaller.nds复制到R4内存卡。
  • 插入R4卡，运行rxinstaller.nds安装 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-11.jpg 400 %} {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-2.jpg 400 %} {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-13.jpg 400 %}

升级虚拟系统到最新

  • 此步骤也可以先略过，不过有可能会安装不了部分游戏因为版本太低，我直接先升级到了最新系统，也可以正常联网。
  • 开机按照 4-1-4方法会进入虚拟系统，最好真实系统和虚拟系统设置不同的主题和图标大小，能更好的分辨，避免在真实系统升级就惨了。
  • 进入虚拟系统后，进入设置查看版本号，确认是RX3D后打开无线升级虚拟系统到新，如果是Ver说明是真实系统千万不要升级。虚拟系统升级过程中最好是闭合3DS升级。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-5.jpg 400 %}
  • 以后下载的游戏都是在虚拟系统中玩的。

安装FBI

  • 更新系统完毕后，继续4-1-4操作进入rxTools界面，长按R进入rxToos菜单。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-7.jpg 400 %}
  • 进入Other Option，选择第二个选项安装FBI，按Y键打算安装在虚拟系统（一般都推荐安装在虚拟系统），会出现ERROR的英文单词，其中第三行英文“TMD Veraion:vXXXX”就是要记住的数字。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-12.jpg 400 %} {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-14.jpg 400 %}
  • 关机取出SD卡，在工具包中rxTools v2.6/fbi_injection/ 下进入对应3DS语言版本的文件夹，然后选择上一步需要记住数字的文件夹，把fbi_inject、fbi_inject.tmd两个文件复制到SD卡根目录，顺便复制一个小游戏，我复制了港版新马里奥兄弟2。
  • 继续开机，利用4-1-4进入rxTools菜单，长按R进入rxToos菜单，重新进入Other Option，选择第二个选项安装FBI，按Y键安装在虚拟系统，安装成功后，重新进入虚拟系统，打开健康安全软件就是FBI软件了（此处帖子也有安装BigBlueMenu.cia进行安装cia的，我没有安装，直接用默认FBI安装cia游戏的） {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-6.jpg 400 %}

可以安装cia游戏下载包了

我一般去电玩巴士下载CIA游戏，把CIA游戏复制到SD卡，打开健康安全软件进入FBI，可以直接选择你复制的CIA游戏进行安装了，安装时间有点慢，等等就好了，现在可以享受众多游戏了，玩不过来啊。 {% img http://7xnr2r.com1.z0.glb.clouddn.com/2015-10-27-3.jpg 400 %}

注意事项

  • 最好不要在虚拟系统插R4烧录卡，会导致无法4-1-4进入虚拟系统。
  • 进入虚拟系统设置后，退出系统设置时为真实系统，这个务必要注意!!!
  • 如果4-1-4无法进入虚拟系统，重新插入R4卡运行rxinstaller.nds即可。
