安装Yosemite

  1. 以下安装步骤请详细参考： tonymacx86  
  2. 登录App Store，下载 OS Yosemite。  
  3. 准备一个8G以上的U盘，用磁盘工具格式化。  
  4. 下载 UniBeast  
  5. 打开UniBeast，安装Yosemite到U盘。  
  6. 下载 MultiBeast，放到制作好的U盘。
  7. 安装系统前的BIOS设置：
  • 如果CPU支持VT-d，关掉它
  • 如果系统有CFG-Lock，关掉它
  • 如果系统有Secure Boot Mode，关掉它
  • 设置系统类型为 Other OS
  • 保存，退出
  1. 开始安装苹果系统，一切正常安装。
  2. 安装后重启，设置帐户，进入系统，准备用MulteBeast安装驱动。
  3. 这里因为我盲目看了参考，安装完系统直接用MultiBeast安装驱动，导致直接用MultiBeast安装驱动后，网卡驱动一直装不上，重新安装后我发现，默认安装后是可以正常上网的，只是缺少了声音驱动。所以安装完系统我建议先查看下系统是否可以正常使用，再来安装MultiBeast。
  4. 此处是我MultiBeast配置：
  • Quick Start =》 DSDT Free 
  • Drivers->Audio->ALC892
  • Disk->3rd Party SATA
  • Disk->TRIM Enabler->10.10.3 TRIM Patch (支持固态硬盘)   
  • Graphics->Inter Graphics Patch for Mixed Configurations
  • 基本这些是主要的，其它可以默认或不选。  
  1. 我在MultiBeast里面没有勾选安装网卡驱动，安装完后还是声音驱动正常了，但网卡又不能使用了，自己手动下载了不少网卡驱动皆不能正常运行。后看到帖子，下载了Kext Utility来安装网卡驱动后，重启后终于能正常使用了。
  2. 安装后己知问题:电脑睡眠或休眠醒来后声音无法正常播放，要重新启动后才能恢复正常。

Mac开发配置

安装Command Line Tools for Xcode

  1. 如果安装了Xcode，可以跳过这一步。
  2. 前往苹果开发者下载页面，用你的AppleID登录。
  3. 可能第一次登录需要回答填写一下问题，完成问题后 进入下载页面，搜索下载: 我下载安装了稳定版 Command Line Tools(OS X 10.10) for Xcode 6.4。

安装以下步骤请参考Mac开发配置手册

  1. 安装Homebrew（OS X系统中最受欢迎的包管理工具）
  2. 安装iTerm2
  3. 安装ZSH
  4. 安装Git
