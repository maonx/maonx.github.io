黑苹果Yosemite直接升级到EI Capitan

2015-11-03

文章目录

    • 制作EI Capitan启动U盘
    • 设置系统BIOS
    • 升级OS X EI Capitan
    • 安装和设置Clover
    • 大功告成！！！！

本机原系统黑苹果Yosemite，准备升级EI Capitan，主要资料参考网站：tonymacx86。

制作EI Capitan启动U盘

  1. 登录App Store下载EI Capitan OS，准备1个16G空间或以上U盘。
  2. 利用系统磁盘工具格式化U盘。
  3. 下载UniBeast。
  4. 系统字体必须为英文才能启动UniBeast，制作EI Capitan启动U盘。

详细安装步骤参考此帖子：UniBeast: Install OS X El Capitan on Any Supported Intel-based PC。

设置系统BIOS

一开始制作好U盘后，我启动利用Clover进入不了系统，一度放弃更新了，后不甘心又在上述参考网站查了资料后，设置BIOS，顺利成功利用Clover进入。

我的主板是Gigabyte AMI UEFI BIOS 9系主板，所以设置会比较简单：

  1. 开机按Del进入BIOS设置。
  2. （重置为默认设置）Load Optimized Defaults。
  3. CPU如果支持VT-d， 关了它（disable）。
  4. 如果系统支持Secure Boot Mode，关了它（disable）。
  5. 设置OS Type为 Other OS。
  6. 保存并退出。

升级OS X EI Capitan

  1. 插入启动U盘，有USB2的话一定要插在USB2接口。
  2. 我在制作U盘时选择的是UEFI，所以开机直接进入启动U盘显示Clover菜单，如果是老式主板，制作时选择Legacy模式，开机后按F12进入启动模式选择从U盘启动。
  3. 进入Clover菜单后，选择启动原系统Yosemite。
  4. 如果顺利利用Clover进入原Yosemite，就可以进入设置以下步骤继续了，如果进入不了，那只能利用启动U盘重新安装EI Capitan了。
  5. 参考此帖子，备份系统，并删除Chimera相关文件。
  6. 现在可以直接点击应用程序中的安装OS X EI Capitan 。 这个很重要，我第一次看错了，先自己安装了Clover，重启后一直进不了系统，还好有备份，然后删除EFI分区里面的内容，重新这一步安装才成功了。
  7. 按安装步骤来，中间需要重启的话，重启，然后还是要利用启动U盘进入Clover菜单，选安装Yosemite那个（默认巻名还是Yosemite），启动后会显示正在安装，剩余时间。
  8. 安装后顺利进入系统，可以看到系统版本已经是10.11了，但现在依然没有结束，因为开机还是要依靠启动U盘。

安装和设置Clover

  1. 下载EI Capitan Post-Install Tools，安装UEFI版本的Clover, 并安装CustoMac Essentials。

  2. 下载 Clover Configurator,并打开，默认此软件会自动挂载EFI分区。

  3. 选择 SMBIOS，点击魔术棒那个按钮，选择一种配置，然后关闭并保存。

  4. 安装声卡驱动，下载CloverALC110 script, 双击运行，在命令行按Y安装。

  5. 设置Clover自动启动EI Capitan。 编辑 /EFI/CLOVER/config.plist

  6. 若找不到/EFI ，可下载 EFI Mounter v2。

大功告成！！！！
